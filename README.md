# End-to-End Data Engineering Project using Microsoft Fabric

## 📌 Overview
This project demonstrates an end-to-end data pipeline including ingestion, transformation, and analytics.

## 🚀 Day 1
- Loaded ecommerce dataset into Lakehouse
- Created raw tables
- Identified data quality issues

## ⚠️ Data Issues
- Duplicates in orders
- Null values in payments
- Negative payment values
- Missing customer IDs
- Inconsistent date formats

## 🧠 Approach
Raw layer stores data as-is. Cleaning will be done in Silver layer.
